# WhatsApp Data Extraction & Optimization Scripts

This folder contains a suite of standalone scripts to extract WhatsApp contacts, links, and high-quality images from chats or channels, and process them for web use. 

The scripts are prefixed with numbers to guide you through the two available workflows: extracting from a **Text Export (TXT)** or extracting directly from **WhatsApp Web**.

---

## The Scripts Overview

### Workflow A: Text Export (Mobile App)
If you exported your chat from the WhatsApp mobile app as a `_chat.txt` file (with media attached):
- **`1_extract_from_txt.py`**: Reads the `_chat.txt`, parses descriptions, links, and dates, and generates the main `data.json`. It optionally copies the images into a temporary folder.

### Workflow B: WhatsApp Web (Channels / Desktop)
If you want to extract images and contacts directly from a WhatsApp Web session using your browser:
- **`1_extract_from_web.js`**: A JavaScript snippet to run in the browser's DevTools console. It auto-scrolls the chat and downloads a `whatsapp_dump.json` containing base64 images and metadata.
- **`2_unpack_web_dump.py`**: Takes the downloaded dump, decodes the base64 images into a dedicated folder, and outputs a clean `extracted_data.json`.
- **`3_merge_data.py`**: Merges the `extracted_data.json` into your main master `data.json`, resolving duplicates and updating existing contacts with newer messages.

### Global Tools
- **`populate_data.sh`**: An automated shell script that executes the entire extraction pipeline. It empties `data.json`, extracts from your mobile TXT backup, and loops through multiple `whatsapp_dump_X.json` web exports, merging everything into one unified, deduplicated database and moving all images to the temp folder.
- **`4_optimize_images.py`**: Takes any folder with images and compresses them into high-performance `.webp` format, resizing them if necessary. Perfect for serving on a website.
- **`5_migrate_to_supabase.py`**: Reads your final `data.json` and automatically uploads the optimized images to your Supabase Storage bucket, inserting the contacts into your `profiles` table as "migrated/seed" profiles (ready to be claimed by users).

### Data Management & Analytics Tools
These scripts help you keep your master `data.json` clean, validated, and enriched:
- **`nsfw_filter.py`**: Uses the modern HuggingFace Vision Transformer to scan all images locally and flags any pornographic or suggestive content.
- **`gender_detector.py`**: Uses the `deepface` library to analyze faces in the images and automatically tags the profile with `"male"`, `"female"`, or `"other"`.
- **`fix_validation.py`**: A powerful sanitization script that normalizes phone numbers (e.g. removes double `+`), strips messy descriptions, renames physical image files if a phone was corrected, and flags every contact with a `"valid": true|false` tag (based on length).
- **`count_valid.sh`**: A quick bash script that reads `data.json` and prints out validation metrics in your terminal: valid vs invalid counts, missing image files on disk, and duplicate phone numbers.
- **`add_country.py`**: Analyzes the international dialing codes (e.g., `+53`, `+52`) of every phone number and appends a `"country"` field (e.g. `"Cuba"`) to the JSON. Also outputs demographic statistics.

## Detailed Usage Instructions

### `1_extract_from_txt.py`
Reads an official WhatsApp `.txt` export, skips videos/GIFs, extracts phone numbers/URLs, deduplicates keeping the most recent message, and saves everything to `data.json`.

```bash
python3 scripts/1_extract_from_txt.py --input "backup/chat/_chat.txt" --output "data.json" --copy-images-to "temp/images"
```

### `1_extract_from_web.js`
1. Open WhatsApp Web.
2. Open DevTools (F12 or Right Click -> Inspect) and go to the Console.
3. Paste the contents of `1_extract_from_web.js` and hit Enter.
4. The script will automatically scroll up. When you are done, run `stopScraping()`.
5. A `whatsapp_dump.json` file will be downloaded.

### `2_unpack_web_dump.py`
Reads the `whatsapp_dump.json` generated by the browser, decodes the images into a folder next to it, and generates the `extracted_data.json`.

```bash
python3 scripts/2_unpack_web_dump.py --input ~/Downloads/whatsapp_dump.json
```
*(Images will be saved to `~/Downloads/whatsapp_dump/`)*

### `3_merge_data.py`
Takes the partial JSON created by the unpack script and merges it into your master `data.json` database. Automatically ignores incomplete phone numbers.

```bash
python3 scripts/3_merge_data.py --input ~/Downloads/whatsapp_dump/extracted_data.json --main scripts/data.json
```
*(Optional: Use `--prepend-folder "whatsapp_dump/"` if you intend to keep images inside subfolders permanently).*

### `populate_data.sh`
This is an automated pipeline script that resets your `data.json` and repopulates it by automatically extracting your Mobile TXT backup and unpacking/merging all 5 web dumps (`whatsapp_dump_1.json` to `5`), centralizing all images into `scripts/temp/images/`.

```bash
./scripts/populate_data.sh
```

### `4_optimize_images.py`
Batch resizes and optimizes images to `.webp` using the `Pillow` library (`pip install Pillow`).

```bash
python3 scripts/4_optimize_images.py --input-dir "scripts/temp/images" --output-dir "public/images" --quality 80
```

### `5_migrate_to_supabase.py`
Uploads your finalized images and contacts directly to your Supabase project as claimable "seed" profiles.
*Requires `pip install supabase`*

```bash
export SUPABASE_URL="https://your-project.supabase.co"
export SUPABASE_SERVICE_KEY="your-service-role-key"
python3 scripts/5_migrate_to_supabase.py --input scripts/data.json --images-dir public/images
```

### `nsfw_filter.py`
Runs a modern HuggingFace Vision Transformer to scan all downloaded images for NSFW/pornographic content and flags them in your JSON, protecting your database from inappropriate content.
*Requires `pip install transformers torch pillow tqdm`*

```bash
python3 scripts/nsfw_filter.py --input scripts/data.json --images-dir scripts/temp/images
```

### `gender_detector.py`
Uses a modern HuggingFace Vision Transformer (`rizvandwiki/gender-classification-2`) to analyze the primary face in each image and classify the person's gender (male/female). If a face cannot be detected, it defaults to "other".
*Requires `pip install transformers torch pillow tqdm`*

```bash
python3 scripts/gender_detector.py --input scripts/data.json --images-dir scripts/temp/images
```

### `fix_validation.py`
Run this whenever you want to sanitize, fix malformed numbers, and flag invalid entries in your database.

```bash
python3 scripts/fix_validation.py --input scripts/data.json --images-dir scripts/temp/images
```

### `count_valid.sh`
Run this for a quick dashboard of your data health. It checks the JSON and the physical image directory.

```bash
./scripts/count_valid.sh
```

### `add_country.py`
Run this to enrich your dataset with country names based on their international prefix.

```bash
python3 scripts/add_country.py --input scripts/data.json
```
